# BaseYield Web

Next.js App Router frontend for BaseYield.

## Env
Copy `.env.example` to `.env.local` and set:
- `NEXT_PUBLIC_CHAIN_ID`
- `NEXT_PUBLIC_VAULT_ADDRESS`
- `NEXT_PUBLIC_USDC_ADDRESS`
- `NEXT_PUBLIC_EXPLORER_BASE_URL`
- `NEXT_PUBLIC_DEPLOYMENT_BLOCK`

## Commands
```powershell
npm run dev
npm run lint
npm run build
```
